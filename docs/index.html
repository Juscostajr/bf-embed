Before / After Slider

Coloque suas imagens em docs/images/ com os nomes before.png e after.png, ou use os controles abaixo para carregar imagens locais.

Uso

  • Para publicar no GitHub Pages: ative Pages configurando a pasta docs/ no branch principal (main/master).
  • Se quiser que as imagens apareçam automaticamente, coloque seus arquivos em docs/images/before.jpg e docs/images/after.jpg.
  • Também é possível carregar imagens locais usando os botões "Before" e "After".
