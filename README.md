Before/After slider for GitHub Pages

Coloque este projeto no GitHub e ative GitHub Pages apontando para a pasta `docs/` no branch principal.

Como usar
- Coloque suas imagens em `docs/images/before.png` e `docs/images/after.png` para que apareçam automaticamente.
- Ou abra `docs/index.html` e use os controles "Before" e "After" para carregar imagens locais.

O arquivo principal da página é `docs/index.html`. O slider é implementado com `docs/styles.css` e `docs/slider.js`.
